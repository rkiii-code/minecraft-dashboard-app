# UI Screens (Shimaenaga Theme)

## テーマ / 配色・トーン
- 背景: 白〜淡いグレー、紙質風パターンを薄く敷く。
- アクセント: 空色 `#b7e1ff`、枝色 `#8b6b4a`。強調は空色の塗り/アウトライン。
- 角丸は大きめ（12–16px）、影はごく薄いドロップシャドウ。
- フォント: 「Noto Sans JP」など癖のないサンセリフ。数字は等幅。
- アセット: しまえながのイラスト/写真をヒーローやカードに小さく差し込み。`src/assets/shimaenaga.png` があれば使用、無ければ淡い鳥シルエットのパターンで代替。
- デフォルトアバター: `src/assets/default.jpg` を使用（ゲーム寄りのドット/ブロック調しまえなが）。アバター未設定時の表示やプレビューで必ず適用。

## 画面一覧
### /login
- 左: ロゴと簡単なコピー。「しまえなが」が枝に止まる小イラスト。
- 右: ログインフォーム（ユーザー名/パスワード）。エラーはカード上部に小さく。
- ボタンは空色フラット、ホバーで薄い影。

### /dashboard
- ヒーロー: しまえながイラスト＋「今のサーバーのようす」コピー。
- ステータスカード: オンライン/オフライン、MOTD、バージョン、最終チェック時刻。
- プレイヤー一覧: オンライン状態バッジ付き。アバターは `default.jpg` フォールバック、UUID から色生成の枠を付けても良い。
- メトリクスランキング: 主要 objective をタブまたはドロップダウンで切替。トップ N（例 5〜10件）をバー/リスト表示。
- レイアウト: 2 カラム（左: ステータス/メトリクス、右: プレイヤー一覧）をブレークポイントで縦積み。

### /players/:id
- ヘッダー: プレイヤー名・UUID・オンライン状態バッジ。最後のログイン時刻。
- スコアタブ: メトリクス別にカード表示。最新値＋簡易スパークライン/履歴テーブル。
- プロフィールカード: 初回ログイン日時、累計プレイ時間など。アバターは `default.jpg` フォールバック。

### /admin/metrics
- 一覧テーブル: objective 名、表示名、単位、状態（有効/無効）、最終更新。
- 操作: 追加（モーダル）、編集（行アクション）、無効化/削除。
- バリデーション: objective 名はユニーク・英小文字/アンダースコアのみを即時チェック。

### /profile（自分のプロフィール編集）
- カバー: しまえながイラスト＋「自分のなまえとひとこと」コピー。
- フォーム: 表示名、ひとこと（140 文字程度）、アバター URL 入力。`default.jpg` を初期表示し、未入力時はそのまま適用。
- 右にプレビューカードを並べ、ゲーム風の枠・ピクセル寄りの背景パターンを使う。
- 保存ボタンは空色。成功時は枝色ボーダーのトースト、失敗時は赤系アラート。
- 公開範囲は固定（誰でも閲覧可、メールは非表示）を明記。

### /users/:id（他ユーザーのプロフィール閲覧）
- 基本情報カード: 表示名、ユーザー名、ひとこと、アバター（未設定なら `default.jpg`）。サーバーロールは非表示またはアイコンのみ。
- アクション: 自分のページの場合のみ「編集へ」ボタンを表示。他人のページではボタンなし。
- 余白を広くとり、しまえながのワンポイントを背景にうっすら配置。

## 共通 UI 要素
- ナビゲーション: 上部バーにロゴ＋メニュー（Dashboard / Players / Admin / Profile）。ログアウトボタンは右端。
- カード: 余白広め、角丸大きめ、背景は白/淡グレー。ヘッダーに薄い枝色ライン。
- バッジ: オンライン=空色、オフライン=グレー。
- チャート: シンプルなバー/ライン、凡例は最小限。色は空色系で統一。

## レスポンシブ / モバイル
- 1 カラムに畳み、ナビはハンバーガー化。
- テーブルはカード化または横スクロール。ランキングはスクロール可能リストに。

## 状態とフィードバック
- ローディング: スケルトンまたは淡いパルス。
- エラー: カード上部に枝色ボーダーのアラート。再試行ボタン。
- 空状態: しまえながの軽いイラスト＋短いメッセージ（例「データがまだありません」）。

## アクセシビリティ
- 色コントラストは WCAG AA 以上を目標（空色の濃淡で調整）。
- フォーカスリングを明示（空色アウトライン）。
